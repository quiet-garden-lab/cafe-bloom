# 背景画像の置き場所

このフォルダに画像を置くと、ゲームの背景として自動で読み込まれる。
ファイルが無い場合は、今まで通り手描きSVGの背景シーンが表示される(壊れない)。

## ファイル名のルール

`テーマ名-画面名.拡張子` で置く。拡張子は jpg / png / webp のどれでもOK。

### テーマ名
| テーマ名 | 世界観 |
|---|---|
| `classic` | クラシック喫茶 |
| `garden` | 宵の庭園 |
| `diner` | ルート66 |

### 画面名
| 画面名 | 画面 | イメージ |
|---|---|---|
| `cafe` | 喫茶店 | カウンター越しの店内(設定画面もこれを使う) |
| `book` | 花帖 | 開いた古い図鑑 |
| `field` | 花畑 | 庭・ガーデンの一枚絵 |

### 例
- `classic-cafe.jpg` → クラシック喫茶テーマの喫茶店画面の背景
- `garden-field.png` → 宵の庭園テーマの花畑画面の背景

画面ごとの画像が無いテーマは、`テーマ名.jpg`(例: `classic.jpg`)を置けば
そのテーマの全画面共通の背景になる。

## 推奨サイズ

横1536×縦1024くらいの横長。スマホでは中央部分が切り出されて表示される
(object-fit: cover)ので、大事なものは画面中央に描いてもらうこと。
